**AVLedger** is a modern, lightweight, and Free Open Source (FOSS) digital logbook specifically designed for **Aviation Maintenance Technicians (AMTs)**.

Built entirely in **Go** through the lens of *"vibe coding"*, AVLedger focuses on stripping away bureaucratic friction, allowing technicians to focus on what truly matters: aircraft safety and airworthiness.

---

## ✨ Key Features

* **KISS Philosophy:** (Keep It Simple, Stupid) A clean, distraction-free interface designed for the hangar, not just the office.
* **Powered by Go:** High-performance backend ensuring the app stays fast and responsive.
* **Direct PDF Export:** Generate professional, ready-to-print PDF logs for official documentation with a single click.
* **Lightning-Fast Backup:** Secure your entire technical history with an ultra-fast database backup system.
* **Vibe-Coded:** Developed with an intuitive, AI-assisted flow to ensure the tool feels right for the end-user.

## 🛠️ Tech Stack

* **Language:** Go (Golang)
* **Environment:** Google Project IDX
* **Core Principle:** Vibe Coding & Minimalist UX
